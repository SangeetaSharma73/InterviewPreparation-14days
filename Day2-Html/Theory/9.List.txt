An ordered list (<ol>) displays items in a numbered or sequential order.

An unordered list (<ul>) displays items using bullets, without any specific order.

Each list contains list items wrapped in <li> tags.


<!-- Ordered List -->

  <li>First</li>
  <li>Second</li>
</ol>

<!-- Unordered List -->
<ul>
  <li>Apple</li>
  <li>Banana</li>
</ul>
